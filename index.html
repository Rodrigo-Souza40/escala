📖 Como conectar ao Google Sheets — Passo a passo
1Abra o Apps Script da sua planilha
Na planilha "Escala dia a dia", clique em:
Extensões → Apps Script
Isso abre o editor de código vinculado à planilha.
2Cole o código abaixo no editor
Apague o conteúdo existente e cole este código completo:
// ── LEITURA (GET) ──────────────────────────────────────────────────────────── function doGet(e) { var ss = SpreadsheetApp.getActiveSpreadsheet(); if (!e || !e.parameter) { return out({values: ss.getSheets()[0].getDataRange().getDisplayValues()}); } var tabName = e.parameter.tab || ss.getSheets()[0].getName(); var s = ss.getSheetByName(tabName); if (!s) return out({error: 'Aba não encontrada: ' + tabName}); return out({values: s.getDataRange().getDisplayValues()}); } // ── CRIAÇÃO DE ABA (POST) ──────────────────────────────────────────────────── function doPost(e) { try { var ss = SpreadsheetApp.getActiveSpreadsheet(); var payload = JSON.parse(e.postData.contents); var tabName = payload.tab; var srcTab = payload.srcTab; var matrix = payload.rows; if (!tabName || !matrix) return out({error: 'Parâmetros tab e rows são obrigatórios'}); // Cria ou limpa a aba destino var s = ss.getSheetByName(tabName); if (s) { s.clearContents(); } else { var src = srcTab ? ss.getSheetByName(srcTab) : null; s = src ? src.copyTo(ss) : ss.insertSheet(tabName); s.setName(tabName); if (src) s.clearContents(); } // Grava todas as linhas de uma vez if (matrix.length > 0) { s.getRange(1, 1, matrix.length, matrix[0].length).setValues(matrix); } // Posiciona a nova aba logo após a aba de origem if (srcTab) { var srcSheet = ss.getSheetByName(srcTab); if (srcSheet) { ss.setActiveSheet(s); ss.moveActiveSheet(srcSheet.getIndex() + 1); } } return out({ok: true, tab: tabName, rows: matrix.length}); } catch(err) { return out({error: err.message}); } } function out(obj) { return ContentService .createTextOutput(JSON.stringify(obj)) .setMimeType(ContentService.MimeType.JSON); }
3Publique como Web App
No editor do Apps Script:
1. Clique em Implantar → Nova implantação
2. Tipo: App da Web
3. Executar como: Eu (seu e-mail)
4. Quem pode acessar: Qualquer pessoa
5. Clique Implantar e copie a URL gerada

A URL tem este formato:
https://script.google.com/macros/s/AKfy…/exec
4Cole a URL em Configuração
Vá em ⚙ Configuração neste sistema, cole a URL do Web App e clique Salvar e sincronizar.

Pronto — o sistema vai buscar os dados diretamente da planilha sem erro de CORS.
✓ Esta abordagem funciona independente de onde o arquivo HTML estiver — local, Drive ou servidor.
5Atenção: nome da aba e republica obrigatória
Verifique o nome exato da aba na planilha (parte inferior do Sheets). Use exatamente igual em ⚙ Configuração → Nome da aba.
Exemplos: Página1 / Escala / 12/06/2026

⚠ Importante: sempre que atualizar o código do Apps Script, clique em Implantar → Nova implantação e cole a nova URL no sistema. "Editar implantação" não atualiza o código em execução.
6Gerar nova aba diretamente na planilha
Com o novo código do Apps Script instalado, o botão "+ Gerar próxima escala" vai criar uma nova aba dentro da própria planilha (ex: "13/06/2026"), logo ao lado da aba atual, com todos os dados do dia seguinte já preenchidos e as folgas marcadas como SUBSTITUIR.

Após criar, o sistema oferece sincronizar automaticamente para a nova aba.
